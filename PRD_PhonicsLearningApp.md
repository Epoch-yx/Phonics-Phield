# 自然拼读学习软件 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
**PhonicWorld** (暂定名)

### 1.2 产品定位
一款面向学龄前及小学低年级儿童（3-8岁）的自然拼读学习平台，通过游戏化学习方式帮助儿童掌握英语单词的发音规律、含义及实际用法。

### 1.3 核心价值主张
- **寓教于乐**：将枯燥的拼读学习转化为趣味游戏
- **循序渐进**：遵循儿童认知发展规律设计学习路径
- **个性化学习**：支持自定义内容，满足不同学习需求

## 2. 用户分析

### 2.1 目标用户
| 用户群体 | 年龄 | 特点 |
|---------|------|------|
| 主要用户 - 儿童学习者 | 3-8岁 | 注意力有限，偏好视觉化、互动化内容 |
| 决策者 - 家长 | 25-45岁 | 关注学习效果、教育价值、使用安全性 |
| 使用者 - 教师/机构 | 22-50岁 | 关注批量管理、内容定制、进度追踪 |

### 2.2 用户场景
- 家长希望孩子在家中通过游戏学习自然拼读
- 教师在课堂或课后辅导中使用软件进行教学
- 儿童自主探索学习，自定义添加感兴趣的内容

## 3. 功能需求

### 3.1 核心功能矩阵

| 功能模块 | 功能点 | 优先级 | 描述 |
|---------|--------|--------|------|
| **游戏学习** | 拼读小游戏 | P0 | 音素识别、字母匹配、拼读组合等游戏 |
| **游戏学习** | 词汇冒险游戏 | P0 | 在场景中根据发音规则拼写单词 |
| **游戏学习** | 发音跟读练习 | P0 | AI评分跟读，纠正发音 |
| **课本中心** | 课本上传 | P0 | 支持PDF/图片格式上传 |
| **课本中心** | 划线发音 | P0 | 选中单词即时发音演示 |
| **课本中心** | 读音范例库 | P0 | 内置标准发音音频 |
| **闪卡学习** | 自定义闪卡 | P0 | 添加单词、音标、例句、配图 |
| **闪卡学习** | 分类文件夹 | P0 | 按主题/难度等自定义分类 |
| **闪卡学习** | 收藏功能 | P0 | 收藏常用/重要闪卡 |
| **个人中心** | 学习进度 | P1 | 记录学习天数、掌握词汇量 |
| **个人中心** | 学习报告 | P1 | 家长/教师查看学习数据分析 |

### 3.2 功能详细说明

#### 3.2.1 游戏学习模块

**拼读小游戏**
- 音素识别：听音选字母/字母组合
- 字母匹配：图片与首字母匹配
- 拼读组合：拖拽字母组成单词
- 设计风格：卡通角色、色彩鲜明、即时反馈

**词汇冒险游戏**
- 岛屿/森林等主题场景
- 每关设定拼读规则目标（如：短元音a）
- 完成拼读任务解锁下一关卡
- 奖励机制：星星、金币、角色装扮

**发音跟读练习**
- 展示单词标准发音视频/音频
- 儿童跟读后AI评分（0-100分）
- 提示常见发音错误
- 跟读历史可回放

#### 3.2.2 课本中心模块

**课本上传**
- 支持格式：PDF、图片（jpg/png）
- OCR文字识别（可选功能，后期迭代）
- 书架式展示已上传课本

**划线发音**
- 长按/选中单词
- 即时播放该单词美式/英式发音
- 可调节语速（0.75x, 1x, 1.25x）
- 显示音标

**读音范例库**
- 内置K-2年级常用词汇发音
- 按年级/主题/拼读规则分类
- 支持离线下载

#### 3.2.3 闪卡学习模块

**自定义闪卡**
- 正向：单词 + 配图
- 反向：音标 + 例句 + 发音
- 支持录音自定义发音
- 批量导入（CSV格式）

**分类文件夹**
- 创建文件夹（如：动物类、食物类、年级词汇）
- 文件夹可嵌套（最多2层）
- 拖拽移动闪卡到文件夹

**收藏功能**
- 一键收藏重要闪卡
- 收藏列表独立展示
- 支持按学习熟练度排序

## 4. 非功能需求

### 4.1 性能要求
- 应用启动时间 < 3秒
- 发音响应时间 < 500ms
- 离线模式下核心功能可正常使用

### 4.2 安全与隐私
- 儿童模式：无广告、无外部链接
- 家长模式：设置使用时长、内容过滤
- 数据加密存储（尤其是用户录音）

### 4.3 平台支持
- iOS 14.0+
- Android 8.0+
- Web端（可选，后期迭代）

### 4.4 无障碍要求
- 大字体、高对比度模式
- 屏幕阅读器兼容
- 操作按钮足够大（适合儿童点击）

## 5. 技术架构（初步建议）

### 5.1 技术选型
| 层级 | 技术方案 | 说明 |
|------|----------|------|
| 前端 | Flutter / React Native | 跨平台开发 |
| 后端 | Node.js / Python | RESTful API |
| 数据库 | PostgreSQL + Redis | 主数据 + 缓存 |
| 文件存储 | OSS / S3 | 课本/音频/图片存储 |
| 语音服务 | 第三方API（如百度/科大讯飞） | 发音评测 |

### 5.2 数据模型（初步）

```
用户 (User)
├── id, name, age, role (child/parent/teacher)
├── 进度数据
└── 闪卡/文件夹关系

闪卡 (Flashcard)
├── id, word, phonetic, example, audio_url, image_url
└── folder_id, is_favorite, mastery_level

文件夹 (Folder)
├── id, name, parent_id, user_id
└── 闪卡列表
```

|--------|------|----------|
